## Modern monolithic pwa with ssr boilerplate

Application with laravel, vue js, vite, inertia js, tailwind css, typescript
  
Run command for development

```
npm run dev
```

for frontend build

```
npm run build
```

for ssr

```
node bootstrap/ssr/ssr.mjs
```
